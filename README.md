Após testar a primeira versão do aplicativo, a cliente ficou satisfeita com o resultado, porém solicitou novos recursos para melhorar a organização da festa.
Para isso, será necessário incrementar o aplicativo, criando novas telas e menus, permitindo ao usuário navegar entre elas de forma simples.

1.	Primeira Tela – Informações do Evento
A primeira tela do aplicativo deverá conter:

•	Nome do evento (ex.: Aniversário de 65 anos);
•	Data do evento.

Esta tela será a tela inicial do aplicativo.
________________________________________
2.	Segunda Tela – Menu Principal
A segunda tela deverá conter um menu de opções, permitindo ao usuário acessar as funcionalidades do aplicativo.

O menu possui as seguintes opções:
•	Menu 1 – Lista de Convidados
Tela para adicionar e visualizar uma lista simples de convidados.
•	Menu 2 – Controle de Gastos
Tela para registrar e visualizar os gastos da festa (ex.: aluguel, comida, decoração).
•	Menu 3 – Cálculo de Comes e Bebes
Tela para auxiliar no cálculo da quantidade de comida e bebida necessária para a festa.
